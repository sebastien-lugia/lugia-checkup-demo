Ce qui est testé ici

Chaque atome est rendu côté React dans web/components/v3/atoms.tsx avec le même contrat visuel que ce que tu vois ci-dessus. Les couleurs viennent de web/lib/v3/tokens.ts via paletteFor(theme) — aucun atome ne code en dur une couleur.

Le composant Em remplace strictement <em> avec deux variantes : argent pour une emphase neutre, warn pour une emphase fonctionnelle (alerte sémantique). Plus jamais d'italique dans V3 — la règle est encodée à trois endroits (reset CSS, fontRules.italicAllowed: false, fontStyle: "normal" explicite dans Em).

Atomes posés : DVMono, DVCorners, LevelBar (avec variantes colorées par axe), EffortBadge (low / med / high), GainBadge (fast / mid / slow), Em (argent / warn), Divider, SurfaceShimmer. Toutes les variantes acceptent theme: "day" | "night".
